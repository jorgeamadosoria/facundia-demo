Facundia Demo

Esta página es un demo para Facundia, una librería para manipulación de palabras en español.

Facundia es un simple jar que puede ser incluida en cualquier aplicación Java que use un gestor de dependencias.

Funcionalidades
  • Escritura de números cardinales: Permite que se escriban números cardinales en su forma textual. Los números escritos siguen el formato de escala larga hasta los cuatrillones, la mayor cantidad reconocida por la Real Academia como una palabra dedicada a nombrar una cantidad numérica.
  • Sílabas: Permite la separacion de palabras en sílabas según un enfoque tónico. Por ejemplo, etimológicamente la palabra desahucio se dividiría en des-ahu-cio, mientras que tónicamente la división silábica es de-sahu-cio.
  • Pluralización: Permite obtener el plural de cualquier palabra en español siguiendo un conjunto de reglas estructurales y una lista de plurales excepcionales. La librería es estructural y no semántica.
  • Conjugación verbal en español: Permite obtener todas las conjugaciones de un verbo. Las expresiones regulares usadas no incluyen análisis semántico de las palabras. Probado con 9000+ verbos existentes en español extraídos del sitio oficial del DRAE. Algunos verbos (como follar, asolar o desolar, engrosar y otros) tienen dos formas, en cuyo caso se genera solo una de los dos en todos los casos. Los verbos defectivos se generan como si todas las formas personas existieran.
Acerca de Facundia

Esta página es un demo para Facundia, una librería para manipulación de palabras en español.

Facundia es un simple jar que puede ser incluida en cualquier aplicación Java que use un gestor de dependencias. Incluye otra librería para separar las palabras en sílabas, mediante un Autómata Finito Determinista (DFA en inglés).
